## О себе

+ Более 6 лет коммерческой разработки на JavaScript/TypeScript и React.
+ Опыт работы с Next.js, управлением состоянием (Redux, MobX) и кросс-браузерной версткой.
+ Глубокая практика тестирования (TDD, Jest, React Testing Library) и оптимизации производительности фронтенд-приложений.
+ Умение работать в кросс-функциональных командах по Scrum, настраивать CI/CD и инструменты разработки (Storybook, Hygen).
+ Продуктовое мышление, направленное на улучшение UX, метрик и создание масштабируемой архитектуры.

## Ключевые навыки

   **Технологии:** JavaScript, TypeScript, HTML5\
   **Фреймворки & Библиотеки:** React, Next.js, Node.js\
   **Управление состоянием:** Redux (RTK), MobX, Effector\
   **Стилизация:** Styled Components, Emotion, CSS Modules, SCSS, CSS3\
   **Тестирование:** Jest, React Testing Library, TDD, Playwright\
   **Инструменты:** Git, Docker, Webpack, REST API, OpenAPI, Swagger, Storybook, Hygen, Jira, Confluence, TeamCity (CI/CD)\
   **Методологии:** Scrum, Agile
